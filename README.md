# Proyecto de Cuidado del Agua

## Descripción

Este proyecto es una página web educativa e interactiva diseñada para enseñar a niños de 6 a 10 años sobre la importancia del cuidado del agua. La página ofrece varios recursos didácticos para que los niños aprendan de una manera divertida y participativa.

## Características

- **Videos Educativos:** Una sección de videos interactivos que enseñan sobre el ciclo del agua, la contaminación y consejos para ahorrar agua.
- **Juegos Interactivos:** Un juego llamado "Atrapa las Gotas" para que los niños aprendan mientras se divierten.
- **Quiz Educativo:** Un quiz en Kahoot para evaluar el conocimiento adquirido sobre el cuidado del agua.
- **Consejos Prácticos:** Una sección con consejos fáciles de seguir para que los niños puedan aplicar en su vida diaria.
- **Recursos Descargables:** Una guía educativa en formato PDF para que los niños puedan seguir aprendiendo sin necesidad de estar conectados a internet.

## Cómo usar

Para utilizar la página, simplemente abre el archivo `index.html` en tu navegador web. Desde allí, podrás acceder a todas las secciones del proyecto.

## Tecnologías utilizadas

- **HTML:** Para la estructura de la página web.
- **CSS:** Para el diseño y la presentación de la página.
- **JavaScript:** Para la interactividad y la funcionalidad de la página.
- **Kahoot:** Para el quiz educativo.
